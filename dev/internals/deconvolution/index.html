Deconvolution

JuChrom.nnlspenalized — Function
nnlspenalized(
    F::AbstractMatrix{<:Real},
    y::AbstractVector{<:Real};
    ridge::Real=1e-12,
    μ=nothing,
    λ=nothing,
    w=nothing
) -> Vector{Float64}

Return the optimal nonnegative coefficient vector a (length Kpeaks) from solving a small nonnegative least squares problem (NNLS) using OSQP, a quadratic programming solver:

minimize_a  ||F*a - y||^2  +  sum_k λ[k]*(a[k] - μ[k])^2
subject to  a >= 0.

The optional quadratic prior is controlled by μ and λ. When provided, it softly pulls the solution toward μ with per-component strength λ. Set λ[k]=0 to disable the prior for component k.

Optional heteroscedastic weights w apply per observation. They are implemented by row-scaling F and y by sqrt.(w), which converts the weighted problem to an equivalent unweighted least squares system.

F is the (n_scans × Kpeaks) design matrix where column k is component k evaluated at the retentions, and y is the length-n_scans observed intensities for one channel. ridge adds a small diagonal regularization for numerical stability.

See also unimodalfit, unimodalfit_apexsearch.

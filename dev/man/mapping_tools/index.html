Mapping tools

The retention mapping API is split by task:

  • Mapper containers documents AbstractRetentionMapper, RetentionMapper, calibration anchors, units, metadata, and domain bounds.
  • Fitting maps documents fitmap and fitting controls.
  • Applying maps documents forward, inverse, derivative, and raw mapping functions.
  • Plotting maps documents the Makie diagnostic plot for fitted mappers.

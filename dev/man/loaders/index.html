JuChrom.AgilentFIDLoader.AgilentFIDLoaderSpec — Type
AgilentFIDLoaderSpec{F}(path::String)

Internal structure representing a configured load request for Agilent FID data. This type encapsulates:

  • the source path, and
  • a type tag for format versioning.

Normally, this is constructed indirectly via AgilentFID(...).

JuChrom.AgilentFIDLoader.AgilentFID — Function
AgilentFID(path::String) -> AgilentFIDLoaderSpec

Construct a loader specification for Agilent FID data. The path can be either a directory (containing FID1A.ch) or a direct path to the file.

Returns a AgilentFIDLoaderSpec object used for deferred data loading via load(...).

JuChrom.ChemStationMSLoader.ChemStationMS — Function
ChemStationMS(path::String; mode::Symbol=:ms) -> ChemStationMSLoaderSpec

Construct a loader specification for Agilent ChemStation GC/MS data. The path can be either a directory (containing data.ms) or a direct path to the file.

The keyword argument mode determines the type of data to load:

  • :ms → full mass spectral data (default)
  • :tic → total ion chromatogram only

Returns a ChemStationMSLoaderSpec object used for deferred data loading via load(...).

JuChrom.ChemStationMSLoader.ChemStationMSLoaderSpec — Type
ChemStationMSLoaderSpec{F}(path::String; mode::Symbol=:ms)

Internal structure representing a configured load request for ChemStation MS data. This type encapsulates:

  • the source path,
  • selected mode (:ms or :tic), and
  • a type tag for format versioning.

Normally, this is constructed indirectly via ChemStationMS(...).

FileIO.load — Function
load(req::MassHunterMSLoaderSpec{MassHunterMSv1}) -> AbstractScanSeries

Loads and parses mass spectrometry data from Agilent MassHunter (version 1) GC/MS files. Takes a MassHunterMSLoaderSpec created via MassHunterMS(...).

Returns a AbstractScanSeries subtype object containing either:

  • a vector of MassScan objects (for mode=:ms), or
  • a vector of ChromScan objects (for mode=:tic),

along with parsed metadata (sample, user, acquisition, instrument).

# Mapping tools

The retention mapping API is split by task:

- [Mapper containers](mapping_containers.md) documents `AbstractRetentionMapper`,
  `RetentionMapper`, calibration anchors, units, metadata, and domain bounds.
- [Fitting maps](mapping_fitting.md) documents `fitmap` and fitting controls.
- [Applying maps](mapping_application.md) documents forward, inverse, derivative, and raw
  mapping functions.
- [Plotting maps](mapping_plotting.md) documents the Makie diagnostic plot for fitted
  mappers.
